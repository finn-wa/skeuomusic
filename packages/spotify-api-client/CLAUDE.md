TypeScript web client for the Spotify API, offering a REST client and an embedded web player.

- `src/auth`: Authentication flow code based on the official Spotify TypeScript SDK.
- `src/openapi`: REST API client generated by OpenAPI Generator. Any manual edits to this code will be overwritten. See `./openapi/generate-client.ts` for the script that generates this code
- `src/player`: Spotify Web Playback SDK. Allows the browser to register as a Spotify playback device. Simply a TypeScript port of the JS script provided by Spotify.
